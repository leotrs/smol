About SMOL

Spectra and Matrices Of Little graphs

SMOL is a comprehensive database of 12.3 million graphs (all simple graphs up to 10 vertices) with precomputed spectral properties for six matrix types: adjacency, Kirchhoff Laplacian, signless Laplacian, normalized Laplacian, non-backtracking (Hashimoto), and non-backtracking Laplacian.

Use it to find examples, explore cospectral pairs, discover switching mechanisms that explain cospectrality, and understand which matrix types can distinguish which graphs.

API

All endpoints return JSON by default. Use Accept: text/html header or access via browser for HTML.

Core Endpoints

# Graph lookup and details
GET /graph/{graph6}                    # Graph details with cospectral mates
GET /graphs?n=7&regular=true           # Query by properties
GET /search?diameter=3&girth=5         # Advanced search
GET /random                            # Random graph
GET /random/cospectral?matrix=adj      # Random cospectral family

# Comparison and similarity
GET /compare?graphs=D?{,DEo            # Side-by-side comparison
GET /similar/{graph6}?matrix=adj       # Spectrally similar graphs

# Mechanisms and statistics
GET /api/graph/{graph6}/mechanisms     # Switching mechanisms for a graph
GET /api/stats/mechanisms?n=9          # Mechanism coverage statistics
GET /stats                             # Database statistics
GET /glossary                          # Terminology reference
