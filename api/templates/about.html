About SMOL

Spectra and Matrices Of Little graphs

SMOL is a comprehensive database of all simple graphs up to 9 vertices with precomputed spectral properties for seven matrix types: adjacency, Kirchhoff Laplacian, signless Laplacian, normalized Laplacian, non-backtracking (Hashimoto), non-backtracking Laplacian, and distance matrix (connected graphs only).

The full database extending to n=10 vertices (12 million additional graphs) is available on request. Public deployment is currently limited to n≤9 due to hosting costs.

Use SMOL to find examples, explore cospectral pairs, discover switching mechanisms that explain cospectrality, and understand which matrix types can distinguish which graphs.

API

All endpoints return JSON by default. Use Accept: text/html header or access via browser for HTML.

Endpoint Description Example
GET /graph/{graph6} Graph details with cospectral mates /graph/D?{
GET /search Query/filter graphs by properties. Full API docs → /search?n=7&tags=bipartite
GET /random Redirect to random graph /random
GET /random/cospectral Random cospectral family /random/cospectral?matrix=adj
GET /cospectral-pairs Get cospectral pairs /cospectral-pairs?matrix=adj&n=8
GET /compare Side-by-side graph comparison /compare?graphs=D?{,DEo
GET /similar/{graph6} Spectrally similar graphs /similar/D?{?matrix=adj
GET /api/graph/{g6}/mechanisms Switching mechanisms for graph /api/graph/GCQvBk/mechanisms
GET /api/stats/mechanisms Mechanism coverage statistics /api/stats/mechanisms?n=9
GET /stats Database statistics /stats
GET /glossary Terminology reference /glossary
